#### 文档 [https://reactnative.cn/](https://reactnative.cn/)

-   环境搭建
    -   node 12.\*
    -   python 2.\*
    -   android studio
    -   jdk 1.8
    -   npm 设置淘宝镜像
    -   java, python,android sdk 环境变量配置
    -   翻墙 或者 配置 android studio 代理
    -   其余看文档！很重要！
-   运行 android 项目

    ```bash

    cd AwesomeProject
    yarn android
    # 或者
    yarn react-native run-android
    # 或者
    npm run android
    ```

-   node 进程窗口中调试

    -   `R R` reload app
    -   `摇手机` 或者 `ctrl + M` 打开调试菜单
